Swarms can be downloaded through the `pip` package manager. 
Follow the steps below to get your environment set up. We also provide methods of Installation
with VENV, Anaconda, and local environment configuration for contributors.

<Steps>
  ### 1st Step
    - Python 3.10 or higher: [Download Python](https://www.python.org/)

  ### 2nd Step
    - Upgrade pip with `pip install -U pip`
  
  ### 3rd Step
    Download Swarms with `pip install -U swarms`
</Steps>

<Check>Verisoning Check: `pip show swarms`</Check>
<Check> To upgrade Swarms Run: `pip install -U swarms`</Check>
